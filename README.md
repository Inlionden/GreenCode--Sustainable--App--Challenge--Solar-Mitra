# Solar Mitra

**GreenCode Sustainable App Challenge**

Solar Mitra is a Flutter application designed to promote sustainable energy solutions, specifically focusing on solar energy adoption. This project is part of the GreenCode Sustainable App Challenge.

## Features

-   **Solar Potential Calculation**: Estimate solar energy potential for your location.
-   **Cost-Benefit Analysis**: Calculate savings and ROI for solar installation.
-   **Vendor Connect**: Find and connect with local certified solar installers.
-   **Education Hub**: Learn about solar technology, government subsidies, and maintenance.

## Getting Started

To run this project locally, you will need to have [Flutter](https://flutter.dev/docs/get-started/install) installed.

### Prerequisites

*   Flutter SDK: `^3.5.1`
*   Dart SDK

### Installation

1.  **Clone the repository**
    ```bash
    git clone <repository-url>
    cd ganesha
    ```

2.  **Install dependencies**
    ```bash
    flutter pub get
    ```

3.  **Run the app**
    ```bash
    flutter run
    ```

## Project Structure

This project follows a standard Flutter project structure:

-   `lib/`: Contains the Dart code for the application.
-   `test/`: Unit and widget tests.
-   `assets/`: Images, fonts, and other static assets.

Key dependencies used in this project:
